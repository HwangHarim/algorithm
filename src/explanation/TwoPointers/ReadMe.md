# 투 포인트 알고리즘

---
### 개념
> 투 포인터 알고리즘은 두 배열에서 검색 시 쉽고 효율적인 방법으로 사용하기 위해 사용

투 포인터, 즉 두 개의 포인터의 시작 위치 배치방법은 2가지가 있다.   
1. 배역의 첫번째 원소와 배열의 마지막 원소에서 시작하는 경우(반대 진행 방향)
2. 둘다 첫번째 원소에서 시작하는 경우(같은 방향)
